# Employee Attrition Analysis Dashboard

## Project Overview
This project analyzes employee data using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts

## Dashboard Includes
- Department-wise Employee Count
- Gender-wise Employee Count
- Department-wise Salary Analysis
- Job Satisfaction Analysis
- Employee Attrition Analysis

## Skills Demonstrated
- Data Analysis
- Dashboard Creation
- HR Analytics
- Data Visualization
